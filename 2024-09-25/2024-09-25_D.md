# 2024-大数相乘

时间限制：1000 ms / 空间限制：65536 KB / 难度：3

## 题目描述

给定两个非负整数 $A, B$，试求 $A\times B$。

## 输入

**本题测试点包含多组数据。**

第一行，一个正整数 $T$（$1\leq T\leq 10$），表示数据组数。

对于每组数据：

第一行，一个非负整数 $A$（$0\leq A\leq 10^{2000}$）。

第二行，一个非负整数 $B$（$0\leq B\leq 10^{2000}$）。

## 输出

对于每组数据：

输出一行，一个整数，表示 $A\times B$。

## 输入样例

    2
    0
    0
    123456789
    987654321

## 输出样例

    0
    121932631112635269
